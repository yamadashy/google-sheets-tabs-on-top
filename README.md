## Development
- Node.js >= v18

### Setup
```
$ yarn
```

### Watch and build
```
yarn dev chrome
yarn dev firefox
yarn dev opera
yarn dev edge
yarn dev safari
```

### Build extension
```
yarn build chrome
yarn build firefox
yarn build opera
yarn build edge
yarn build safari
```

### Lint codes
Lint only.
```
yarn lint
```

Lint codes with fix.
```
yarn lint:fix
```

### Type Checking
```
yarn type-check
```

### Show supported browsers
```
yarn supported-browsers
```
