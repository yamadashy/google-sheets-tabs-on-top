## Development
- Node.js >= v23.6.0

### Setup
```
$ npm install
```

### Watch and build
```
npm run dev chrome
npm run dev firefox
npm run dev opera
npm run dev edge
npm run dev safari
```

### Build extension
```
npm run build chrome
npm run build firefox
npm run build opera
npm run build edge
npm run build safari
```

### Lint codes
Lint
```
# eslint, tsc --noEmit
npm run lint
```

### Show supported browsers
```
npm run supported-browsers
```
